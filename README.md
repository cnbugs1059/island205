# h2m

Tool for converting HTML to Markdown.

online converter: http://island205.github.io/h2m/

![online converter](./images/online-converter.png)

## Install

```bash
$npm install h2m
```
## How to use

```javascript
var h2m = require('h2m')

var md = html('<h1>Hello World</h1>')
// md = '# Hello World'
```
